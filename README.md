The goal of this project was to move away from the "flat" look of modern web design and return to something that feels rich, tactile, and grounded in tradition. By blending a deep, imperial color palette with classical typography, I’ve created a "Coming Soon" experience that feels like an invitation to a palace.
HTML5 – Structured for clarity and accessibility.
CSS3 – Utilizing Flexbox for perfect centering and CSS Gradients for that metallic gold finish.
JavaScript – A lightweight script for form submission feedback (no heavy libraries needed).
